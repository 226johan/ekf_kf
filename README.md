## 1.使用

- sudo apt install gnuplot
- mkdir build 
- cd build
- cmake ..
- sudo make


## 2. Dependencies依赖

- **系统**
  - Ubuntu >= 20.04
  - Cmake >= 3.0
- **库**
  - gnuplot-iostream
  - eigen
  - vector
  - random
